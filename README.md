# Java Swing Projects Repository

A collection of Java applications built using Swing for graphical user interfaces (GUIs).

## Projects

| Project | Description |
|---------|-------------|
| **Average** | Calculates and manages averages |
| **BurgerSales** | Burger shop sales management system |
| **Interviews** | Student interview management |
| **InterviewStudents** | Resources for student interviews |
| **NStudents** | Student records management |
| **SelectionStudents** | Student selection application |
| **TeacherSalaryCalculator** | Teacher salary calculation tool |

## Usage

1. Navigate to the project folder
2. Compile: `javac *.java`
3. Run: `java MainClass`

## Requirements

- Java JDK 8 or higher

Happy coding!
